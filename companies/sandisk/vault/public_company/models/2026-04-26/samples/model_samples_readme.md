# SNDK Model Samples

Generated from the current SNDK public-company vault on 2026-04-26.

## Files

1. `sndk_compact_dcf_comps.xlsx`
   - Small, fast review workbook.
   - Best for first-pass company triage.
   - Sheets: Summary, Historical, DCF, Comps, Sources.

2. `sndk_scenario_architect.xlsx`
   - Base/Bear/Bull scenario workbook inspired by the SkillsMP architect/challenger pattern.
   - Best for choosing the final canonical model structure.
   - Sheets: Dashboard, Assumptions, Historical, Forecast, DCF, Sensitivity, Checks, Sources.

3. `sndk_comps_focus.xlsx`
   - Relative-valuation workbook with peer multiples and implied value range.
   - Best when comps matter more than DCF.
   - Sheets: Comps Dashboard, Peer Table, Implied Value, Sources.

4. `sndk_challenger_pack.xlsx`
   - Companion audit workbook, not a valuation model.
   - Best for reviewing whether data/model quality is good enough.
   - Sheets: Challenger, Sample Comparison, Sources.

## Current Notes

- All samples are draft models because assumptions are auto-default, not user-approved.
- The DCF-derived EV is far below current public-fallback EV, so the challenger pack flags this gap for explanation.
- The samples are intended for choosing workbook structure and workflow style, not for making an investment decision.
