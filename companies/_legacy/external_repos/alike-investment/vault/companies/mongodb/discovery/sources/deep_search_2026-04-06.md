# MongoDB (MDB) — Deep Search

## 公司概况
- **总部**: 美国纽约，Nasdaq上市 (MDB)
- **创立**: 2007年 (by Dwight Merriman, Eliot Horowitz, Kevin Ryan)
- **CEO**: CJ Desai (2025.11起); 前CEO Dev Ittycheria (2014-2025.11, 11年)
- **业务**: 开发者数据平台，核心产品MongoDB Atlas (云数据库)
- **员工**: ~5,000+人

## 商业模式
- **MongoDB Atlas**: 云数据库即服务(DBaaS)，占总收入75%
- **Enterprise Advanced**: 自管理企业版
- **Community Server**: 开源版本（漏斗顶端获客）
- 收入模型: 订阅+消费混合；Atlas按使用量收费

## AI战略
- MongoDB定位为AI应用的数据基础层
- Vector Search集成（嵌入式向量搜索）
- Voyage AI模型集成（5个Voyage 4系列embedding模型）
- 3000+客户使用Elastic的AI用例
- 2700+客户在Atlas上跑vector database workloads
- Automated Embedding for MongoDB Vector Search (public preview)

## 竞争格局
- vs AWS DynamoDB/DocumentDB: 云厂商自有产品竞争
- vs PostgreSQL/MySQL: 传统关系数据库
- vs Redis, Elasticsearch: 特定workload竞争
- 护城河: 开发者生态 + NoSQL领导地位 + Atlas平台锁定

## 关键风险
1. CEO过渡期风险
2. 云厂商竞争（AWS/Azure/GCP自有数据库）
3. 增速放缓（22%→19%）
4. 开源竞争（license变更争议）
5. 宏观环境影响企业spending
