# Mobvista/汇量科技 (1860.HK) D1-D7 组织穿透

> **重要说明**：Mobvista是港股小盘科技股（690-1000员工），公开可获取的组织信息远少于Fortinet等美股公司。多个维度只能基于有限数据做推断性评估，需标注⛔。

## Step 0：业务本质画像

### Q1: Mobvista在造什么？

**输入**：移动App开发者的用户获取需求 + 全球碎片化的移动流量
**输出**：AI驱动的程序化广告匹配平台（Mintegral）+ MarTech SaaS工具矩阵
**价值创造核心环节**：

| 环节 | 描述 | 收入占比 |
|------|------|---------|
| Mintegral程序化广告 | AI竞价+广告匹配+流量变现 | 95.8% |
| MarTech SaaS | XMP/AdsPolar/GameAnalytics/SolarEngine | ~4.2% |

**收入结构（FY2024: $1.51B，TTM: $2.05B）**：
- 广告技术（AdTech）：$1.45B，占95.8%——Mintegral是绝对核心
- 营销技术（MarTech）：~$60M，占~4%——SaaS产品矩阵尚小
- 毛利率21.0%（FY2024）——典型的广告中介平台利润结构（vs AppLovin 87.9%）
- 营业利润率2.8%（FY2024）——刚转正盈利，利润率极薄

**业务本质定义**：Mobvista的本质是**移动广告领域的AI匹配引擎**。核心能力是用算法将广告主的获客需求与全球碎片化流量高效匹配。这是一个**"规模+算法"双飞轮**的业务：流量规模越大→数据越多→AI模型越准→广告效果越好→更多广告主→更多流量接入。

**与AppLovin的本质区别**：AppLovin同时拥有游戏工作室（已出售大部分）和广告引擎，其AXON引擎掌握一手用户行为数据。Mobvista是纯中介——不拥有内容，依赖第三方流量。这意味着数据飞轮的起点不同。

### Q2: 环境特性

| 维度 | 特征 |
|------|------|
| 竞争格局 | 寡头化——AppLovin(27-37%) > AdMob(33%) > Mintegral(11-16%) > Unity |
| 技术变化速度 | 极快——隐私政策（IDFA/ATT）、AI竞价算法持续迭代 |
| 客户决策周期 | 短——广告投放决策以天/周为单位 |
| 反馈周期 | 极短——广告效果在小时/天内可见 |
| 市场增长 | CAGR 12.9%（程序化广告2026-2035） |
| 地缘因素 | 中国出海生态（中概股风险+中国开发者需求） |

### Q3: 理想组织画像

基于"AI广告匹配引擎"的业务本质，理想组织应该具备：

1. **顶级AI/ML工程团队**：广告竞价的核心是算法，需要最强的ML人才
2. **数据驱动的快速迭代**：反馈周期极短（天级），组织必须能快速实验和部署
3. **全球化运营能力**：覆盖130个国家，需要本地化运营+全球统一技术栈
4. **精简高效**：广告中介利润率薄（毛利率21%），人效必须极高
5. **产品多元化能力**：从AdTech向MarTech SaaS扩展需要产品创新能力

### Q4: 理想vs现实的初步Gap评估

- 精简高效 → **匹配**：人均产出$2-3M，效率极高（690人做$2B+收入）
- 全球化运营 → **匹配**：17个城市办公室，130个国家覆盖
- AI/ML团队 → **部分匹配**：AI竞价贡献>80%收入，但人才密度vs AppLovin未知
- 快速迭代 → **信息不足**：缺乏产品迭代速度的证据
- 产品多元化 → **Gap明显**：MarTech仅占4%，远未达到平台化

## Step 1.5：核心机制识别

### A类（核心机制）

**A1: AI智能竞价引擎**
- 必要性：广告平台的核心竞争力=匹配效率，AI竞价贡献>80%收入
- 因果链条：业务本质="AI匹配引擎"→算法质量直接决定广告效果→决定收入
- 可验证：Mintegral从11%市场份额增长到16%（2025年Sensor Tower数据），增速最快

**A2: 中国出海生态位**
- 必要性：中国开发者是Mobvista的核心客群（阿里、腾讯等），去掉这个定位则无差异化
- 因果链条：中国出海趋势→开发者需要海外获客→Mintegral作为"桥梁"
- 可验证：中国开发者客户名单（Voodoo/iDreamSky等），总部迁至新加坡强化国际化

### B类（支撑机制）
- Meta Adtech badge合作（XMP/AdsPolar获得优先API访问）
- 极高人效（690人做$2B+收入）
- GameAnalytics游戏分析SaaS（80,000+游戏接入，但收入占比微小）

**缺失的取舍**：
- 没有看到明确的"放弃什么"——是否应该放弃某些低利润率的流量？
- 出售Mintegral的尝试暴露了管理层对"持有vs变现"的犹豫
- 与AppLovin的差距（3倍份额差）如何追赶？没有看到清晰的策略

**评分理由**：3分。All-in Mintegral是清晰的聚焦，非游戏扩展是正确方向。但战略取舍的"舍"字不够明确，出售核心资产的尝试暴露了战略不坚定。[S3]
