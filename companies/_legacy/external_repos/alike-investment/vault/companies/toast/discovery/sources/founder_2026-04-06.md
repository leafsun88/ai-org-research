# Toast — Founder/CEO Profile

## CEO: Aman Narang（Co-Founder）
- **背景**: MIT CS → Endeca (Oracle) → 创立Toast
- **创业**: 2013年与Steve Fredette、Jonathan Grimm共同创立
- **升任CEO**: 2024年1月（从President升任）
- **领导风格**: 耐心、注重对齐、客户导向
- **招聘哲学**: 深挖简历中的一件事，理解候选人如何思考和解决复杂问题

## CEO认知特征
1. **垂直深度信仰**: 深耕餐饮一个垂直领域，不追求泛行业
2. **平台思维**: 从POS → 支付 → 运营 → 金融 → AI的层层扩展
3. **长期野心**: "我们有信心构建一个可以服务现有规模数倍餐厅的平台，在未来十年扩展到$5B和$10B ARR"
4. **数据驱动**: 强调用数据建立竞争壁垒（CFO Gomez推动的"data-driven moat"叙事）
5. **纪律性增长**: "投资有纪律" — 不盲目扩张

## 关键语录
- "$5B和$10B ARR目标在未来十年"
- "耐心、建立对齐、贴近最重要的声音——团队和客户"
