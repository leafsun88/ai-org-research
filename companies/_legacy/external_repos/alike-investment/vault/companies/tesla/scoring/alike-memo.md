# Tesla (TSLA) — Alike Memo

## Alike Score: 56/100 | Fit: 2/5

## 一句话结论

Tesla是科技史上最极端的"创始人依赖"组织：Musk的第一性原理思维和技术判断是真实资产，但注意力分散（6家公司+政治）、无继任机制、无原创组织机制，导致组织生成力严重依赖单一变量——而这个变量正处于被稀释状态。**最大亮点**：Key Bet多元且愿景宏大（能源/FSD/Optimus），能源业务已出现真实拐点。**最大短板**：组织缺乏自运行能力，CEO-as-single-point-of-failure是系统性风险，不是周期性噪音。

## 业务本质 → 理想组织

**业务本质**：Tesla是**重资产制造×AI软件×能源基础设施**的三元复合体。

- 汽车制造（76%收入）：重资产、规模效应、毛利率承压（~15%）
- 能源存储（12%收入）：Megapack高增长，31%毛利率，真实盈利亮点
- FSD/Robotaxi（0%收入，大量期权价值）：数据飞轮逻辑，但L4审批待获
- Optimus机器人（0%收入）：2026年5万台量产目标，商业化路径仍不清晰

**理想组织形态推导**：
- 制造业务需要 → 极强工程执行力 + 跨业务线资源协调系统
- AI/软件业务需要 → 创新自由度 + 数据飞轮团队稳定性
- 能源业务需要 → 独立运营能力 + 长周期资本配置判断
- 三元协同需要 → 清晰的组织边界 + 系统化信息架构

**核心矛盾**：业务复杂度需要"多业务线高效协调系统"（类NVIDIA One Architecture），实际运作模式是"一个超人CEO的个人工作台"。当CEO专注时（2018-2023），系统产出惊人结果；当CEO分心时（2025年DOGE期间），利润暴跌71%，组织立即暴露脆弱性。
