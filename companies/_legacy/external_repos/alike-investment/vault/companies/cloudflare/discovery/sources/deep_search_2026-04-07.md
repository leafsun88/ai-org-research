# Cloudflare (NET) — DeepSearch 2026-04-07

## 公司概况
- **创立**: 2009年，Matthew Prince, Michelle Zatlyn, Lee Holloway (Harvard Business School项目)
- **上市**: 2019年NYSE
- **定位**: 全球最大的边缘计算/安全/开发者平台——"Connectivity Cloud"
- **市值**: ~$45B+（2026年4月）
- **员工**: ~4,500+

## 核心业务
- **安全**: DDoS防护、WAF、Bot管理、Zero Trust——每天阻止2150亿次网络威胁
- **性能**: CDN、DNS、负载均衡——全球330+城市节点
- **开发者平台**: Workers(serverless)、R2(对象存储)、D1(数据库)、Durable Objects(有状态)
- **AI**: Workers AI(边缘推理)、AI Gateway、AI Security for Apps
- **网络服务**: Magic Transit、Magic WAN、WARP VPN

## 财务数据（FY2025）
- 全年营收: $2.17B（YoY +30%）
- FY2026指引: $2.785-$2.795B（YoY +28-29%）
- 毛利率: 77%
- Non-GAAP营业利润率: ~15%
- 付费客户: 332,000+（38% Fortune 500）
- 大客户增长: $100K+客户3,756家（+28%）
- 开发者: 450万活跃人类开发者

## Agentic Internet战略
- Prince定位: "如果AI代理是互联网的新用户，Cloudflare就是它们运行的平台和通过的网络"
- Workers AI: 边缘AI推理，低延迟
- Durable Objects: 为AI代理提供有状态内存
- 2026年3月推出"AI Security for Apps"——AI代理安全层
- 40%互联网流量已由自主AI代理生成（2025年中数据）

## 开发者平台
- Workers: Serverless计算——竞争者AWS Lambda@Edge但更易用
- R2: 零出站费用对象存储——直接挑战AWS S3
- D1: 边缘SQLite数据库
- Queues/Pub-Sub: 消息队列
- Containers: 2025年新增——重工作负载支持
- 整体愿景: 完整的云平台，不仅仅是CDN

## 竞争格局
- AWS/Azure/GCP: 传统云巨头——规模远超但开发者体验较差
- Fastly: 边缘计算直接竞争但规模差距大
- Akamai: CDN老牌但创新较慢
- Zscaler/CrowdStrike: 安全垂直竞争
- 差异化: 全球边缘网络+开发者平台+安全一体化+零出站费

## 关键风险
- 估值极高: 27x revenue, 200x+ EPS——需要near-perfect执行
- 规模与hyperscaler差距巨大
- 安全事件风险（2023年曾有targeted attack）
- 从安全/CDN到完整云平台的跨越是巨大挑战
- 竞争加剧——AWS/Azure持续改善边缘能力

## 近期催化剂
- Agentic Internet定位——AI代理需要边缘基础设施
- 开发者平台持续扩展
- 大客户加速增长（$100K+客户+28%）
- 营业利润率持续改善
