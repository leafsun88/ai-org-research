# TransDigm 创始人/CEO 研究

## Nick Howley — 创始人 & 董事长

### 背景
- 1993年创立TransDigm（时名TD Holding Corporation）
- 合伙人：Douglas Peacock
- 初始投资$10M，通过Kelso & Company杠杆收购IMO Industries旗下4家航空零部件公司
- 此后经历三家PE所有者（Kelso → Odyssey → Warburg Pincus），2006年IPO

### 核心理念（来源：50X Podcast / Colossus 四集深度访谈）
1. **IRR门槛**："如果5年不能翻倍，我们不做" — 即最低20% IRR
2. **私有化思维**：将上市公司当PE portfolio运营
3. **去中心化**：决策下沉到最接近价值流的点
4. **资本配置优先**：总部核心功能是资本配置，不是运营干预
5. **关注sole-source + aftermarket**：只收购有专有产品和强后市场的资产

### 公开发言关键信息
- 28年compound annual return 37%（$100K → $175M）
- 前13年PE阶段IRR 37%，后15年上市阶段持续
- IPO以来跑赢S&P 15倍，跑赢同业9倍
- 在国会听证会上就Pentagon定价问题作证（2022年）

### 创始人认知评估
- **思考深度**：高。从"什么值得买"的第一性原理出发（sole-source = pricing power = 可预测FCF）
- **战略定力**：极高。30年坚持同一套收购+运营公式
- **组织设计意识**：强。主动设计去中心化+期权激励体系
- **风险**：可能过度依赖定价权，忽视客户关系长期影响

## Mike Lisman — 新任CEO

### 背景
- Notre Dame航空工程学士 + Harvard MBA
- 2015年前在Warburg Pincus（正是TDG的前PE股东）
- 2015年加入TransDigm，从BU Manager（Aero Fluid Products）做起

### 晋升路径
BU Manager → M&A Lead → CFO/EVP → Co-COO → CEO

### 关键特质
1. **PE + Ops双栖**：在Warburg Pincus做过PE，在TDG做过一线BU运营
2. **M&A能力**：曾主导TDG的M&A pipeline
3. **财务背景**：做过CFO，理解资本结构
4. **内部培养**：10年从BU到CEO，深度理解TDG文化

### 风险
- 管理者型CEO vs 创始人型CEO的差异
- 需验证是否能维持Howley建立的PE文化+收购纪律
- 航空工程背景可能对产品端有更深理解

## Kevin Stein — 即将卸任CEO

- 将过渡为顾问（至2026.3）+董事会成员
- 在任期间TDG从约$200亿市值增长到$740亿+
