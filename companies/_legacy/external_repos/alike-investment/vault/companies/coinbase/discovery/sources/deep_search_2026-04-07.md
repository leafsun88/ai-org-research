# Coinbase (COIN) — Deep Search

## 公司概况
- 成立：2012年，Brian Armstrong和Fred Ehrsam联合创立
- 总部：Remote-first（前SF总部）
- 上市：2021年DPO（NASDAQ: COIN）
- 定位："Everything Exchange" — 加密货币交易所+区块链基础设施+稳定币发行

## 商业模式（多元化）
1. **交易手续费**：现货+衍生品交易（仍是最大收入源但占比下降）
2. **订阅与服务**：$2.8B（2025），同比+23%，占总收入~39%
   - Staking收入
   - USDC利息收入（与Circle分成）
   - Coinbase One订阅
   - 托管服务
3. **Base L2**：以太坊L2网络，稳定币市值达$5.2B
4. **USDC**：2025年稳定币收入$1.35B（占总收入19%）

## 2026展望
- Q1订阅/服务收入指引：$550-630M
- 战略重点：Everything Exchange + USDC扩展 + on-chain活动增长
- "第二波"机构资金进入：从投机转向收益（yield-focused）

## 竞争格局
- CEX竞争：Binance（全球最大）、Kraken、OKX、Bybit
- DeFi竞争：Uniswap、dYdX等去中心化交易所
- 稳定币竞争：Tether(USDT)、PayPal(PYUSD)
- L2竞争：Arbitrum、Optimism、zkSync
- 独特优势：美国最大合规加密交易所+USDC+Base L2垂直整合

## 关键风险
1. **加密周期性**：收入严重依赖crypto市场行情
2. **监管不确定性**：CLARITY Act可能影响stablecoin收益产品
3. **竞争加剧**：Binance+DeFi+新进入者
4. **集中度风险**：交易收入仍占大头
5. **USDC收入依赖利率**：Fed降息将压缩USDC利差收入
