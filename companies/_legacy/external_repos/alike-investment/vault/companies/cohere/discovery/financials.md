## 商业模式
- **核心模式**: Enterprise AI platform — API + 私有部署 + SaaS工作台(North)
- **收入来源**:
  1. **API调用收入**: Command/Embed/Rerank模型的API计费
  2. **North平台订阅**: 企业AI工作台SaaS收入
  3. **Model Vault部署**: 私有化部署许可与支持服务
  4. **云合作分成**: 通过Oracle Cloud、AWS等分发获得收入
- **定价差异化**: 强调推理成本效率——Command R可在2个GPU上运行（对比DeepSeek的16个GPU），降低企业部署门槛
- **部署模式**: Cloud API / VPC隔离 / On-premise / 混合模式
