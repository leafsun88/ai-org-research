# Cursor 3 — Agent-First Workspace Launch

**发布日期**: 2026-04-02
**类型**: Product Launch / Official Announcement
**来源**: [AllBlogThings](https://www.allblogthings.com/2026/04/anysphere-launches-cursor-3-an-agent-first-workspace-integrated-into-its-ai-code-editor.html), [DEV Community](https://dev.to/devtoolpicks/cursor-3-just-launched-with-an-ai-agents-window-what-changed-and-is-it-still-worth-it-496f)

## Key Messages

### Cursor 3: Agent-First重新设计
- 完全重新设计的IDE，围绕**多AI Agent管理**构建
- 工作流从传统代码编辑转变为：指挥、监控、审查自主Agent
- Agent可处理feature实现、测试、部署全流程

### 核心新功能
1. **Agents Window** — 多Agent并行运行面板（local、worktree、SSH、cloud），不打断主编辑
2. **Background Agents** — 云端VM异步执行任务，创建独立分支，完成后生成visual demo
3. **Local-to-Cloud Handoff** — 本地任务无缝迁移到云端
4. **Design Mode** — 视觉UI反馈，直接在编辑器中看到界面效果
5. **Automations** — 外部服务触发的自动化工作流

### 战略意义
- 标志着Cursor从"AI-assisted编辑器"进化为"Agent编排平台"
- 与GitHub Copilot Coding Agent直接竞争
- 强化了Cursor在agentic coding领域的领先地位
