# Nu Holdings (NU) — 组织穿透 D1-D7

## Step 0: 业务本质画像

**业务模型**: 拉美数字银行平台。通过移动端为个人和中小企业提供信用卡、贷款、投资、保险等全栈金融服务。零网点、纯数字化运营。[S5]

**收入结构** [S5·FY2025]:
| 指标 | 数值 |
|------|------|
| FY2025总收入 | $16.3B（+45% YoY）|
| 净利润 | $2.9B |
| 净利润率 | ~18% |
| ROE | 33%（历史新高）|
| 效率比 | 19.9% |
| 客户数 | 131M |
| ARPAC | $15/月 |
| 成本/客户 | $0.80/月 |
| 员工数 | ~9,000 |
| 人均营收 | ~$1.8M |

**理想组织形态**: 数字银行需要：(1) 极强的技术能力（AI/ML信用模型、移动端体验、数据平台）；(2) 极低成本运营能力（效率比<25%）；(3) 跨市场监管能力（巴西+墨西哥+哥伦比亚+美国）；(4) 快速产品迭代能力（新金融产品上线速度）。

## D1: 创始人/CEO认知层次 — 4.0/5

David Velez（44岁）是拉美最杰出的科技金融创始人。Stanford x2 → 投行 → VC(Sequoia) → Nubank。战略核心12年不变：用技术重塑银行服务。[S4]

Responsibility Framework体现了从founder到CEO的认知进化——"建系统而非做事"。Giving Pledge签署+VelezReyes+慈善平台展示长期视野。[S4]

扣分点：CEO 2025年8月卖出3300万股，虽为资产规划但规模不小。super-voting结构给予过大控制权。未经历大经济周期考验。

**证据**: 131M客户，$16.3B收入，33% ROE，效率比19.9%，获客成本<$5 [S5]

## D2: 战略清晰度与执行一致性 — 4.5/5

## D3: 组织架构与决策机制 — 3.5/5

**架构特征**:
- 创始人CEO+Chairman（Velez），super-voting shares [S5]
- 三位联合创始人均在核心位置（Velez/Junqueira/Wible）[S4]
- CTO刚换人（Eric Young，2025年8月加入，前Snap SVP）[S4]
- 前巴西央行行长Campos Neto任Executive Vice-Chairman [S4]
- ~9,000员工 [S3]

**亮点**:
- Campos Neto加入——监管关系的最强背书
- 创始团队稳定——三位联合创始人12年未散
- Eric Young（前Snap SVP）加入——大规模移动工程经验

**风险点**:
- CTO刚换（2025年8月），过渡期执行风险
- super-voting结构——CEO控制过度集中
- 部分团队文化问题（Glassdoor：Data/BA团队toxic）
- 从9,000人管理131M客户→如何扩展到全球？组织扩展压力大
