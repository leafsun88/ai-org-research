# Block Inc — D1-D7 组织穿透

## Step 0: 业务本质画像

**Block在造什么？**
一个连接个人金融(Cash App)和商户支付(Square)的双边金融生态，同时押注Bitcoin作为long-term monetary infrastructure。

**这个业务需要什么样的组织？**
- 极高工程效率（支付零容错）
- 产品间协同（Cash App↔Square↔Afterpay）
- 快速迭代（fintech竞争激烈）
- 长期主义（Bitcoin是decade bet）
- 合规和风控能力

**理想组织形态：**
高效扁平+工程文化驱动+产品协同机制+长期conviction capacity

---

## D1: 创始人/CEO认知与领导力 — 4.0/5

**从业务本质倒推组织形态？** 是。Dorsey的"Intelligence Over Hierarchy"直接回应了"fintech需要极高信息流速度"这个业务本质需求。AI替代中层管理不是跟风，而是对"支付业务中层管理impede information flow"的结构性回答。

**核心证据：**
- 亲自设计IC/DRI/Player-Coach三类角色模型
- Goose AI系统作为信息层（T5T的AI版本）
- $2M GP/employee的极端效率目标
- Bitcoin Blueprint的long-term conviction

**减分项：**
- Bitcoin ideology可能blind to alternatives
- 40%裁员的执行方式引发employee backlash
- 继任计划缺失
- 历史上Twitter/Block注意力分散

## D2: Key Leader配置 — 3.5/5

**最强的人在做最重要的事？** 部分是。

**正面：**
- Amrita Ahuja (COO/CFO双职)说明精简高管层
- Brian Grassadonia领导Cash App——核心增长引擎的leader到位
- Jesse Dorogusker领导Bitcoin硬件——战略性岗位有专人

**问题：**
- 40%裁员后的关键人才流失状况不明
- 中层管理被废除后，DRI机制是否有效运转？
- Jim McKelvey（联合创始人）在战略层但日常参与度不清
- 没有明确的"二号人物"或继任者

## D3: 人才密度 — 3.0/5

**信号：**
- Goose AI处理90%代码提交——这减少了对大量工程师的需求
- 裁员40%后，remaining team是"survivors"还是"believers"？
- Glassdoor 3.7/5（较低），career opportunities 3.3/5
- AI token usage被dashboard追踪——talent管理方式controversial
- 薪酬和RSU被认为good

**关键疑问：**
- ⛔ 缺少直接数据评估remaining team的talent density
- AI mandate是否赶走了top talent？
- "awkwardly human"的感觉是否吸引到了right type of talent？

## D4: 文化机制 — 3.5/5

**Block的文化：**
- 2026前：以"purpose"（economic empowerment）为核心
- 2026后：AI-first + Bitcoin-first + lean machine
- 文化转型幅度极大——相当于文化革命

**正面机制：**
- AI Intelligence Layer客观化了工作分配和追踪
- DRI制度明确了accountability
- Bitcoin Blueprint给公司一个clear north star

**问题：**
- 文化不是codified system（不像Netflix的Culture Deck）
- More mandate than culture-building
- Employee backlash表明文化尚未内化
- 从"economic empowerment"到"AI+Bitcoin machine"的narrative disconnect

## D5: 组织-业务适配度 (Fit Score) — 3.5/5

**适配分析：**

| 业务需求 | 组织回应 | 适配度 |
|---------|---------|--------|
| 支付零容错 | AI代码+精简团队 | ⚠️ 未验证AI code quality |
| 产品间协同 | 扁平结构+DRI | ✅ 有利于协同 |
| 快速迭代 | AI-first+lean | ✅ 理论上更快 |
| 长期Bitcoin bet | 文化统一到BTC | ⚠️ 过度集中风险 |
| 合规风控 | ❓ | ⛔ 裁员后合规团队状况不明 |

**独立评价：** 组织设计方向正确（扁平+AI），但执行风险高。40%裁员的transition cost尚未fully materialized。

## D6: 信息流动与决策质量 — 4.0/5

**正面：**
- 废除中层管理直接改善信息流
- Goose AI系统提供real-time信息
- Dorsey通过AI dashboard直达ground truth
- DRI制度明确决策权

**问题：**
- AI过滤后的信息是否有blind spots？
- 没有"diverse input"机制（不像Netflix的Informed Captain需要seek dissent）
- 决策过度集中在Dorsey一人？

## D7: 组织进化能力 — 4.0/5

**证据：**
- Square→Block品牌重塑
- 从传统fintech→AI-first+Bitcoin machine的radical pivot
- 40%裁员+组织重构展现了dramatic evolution capacity
- 从dual-CEO(Twitter)→focused CEO的个人进化

**问题：**
- 进化的sustainability——这次transformation能持续吗？
- 组织是否有self-correction mechanism如果AI strategy fails？
- 进化速度过快可能导致organizational fatigue
