# SHEIN Substack Analysis Summary

## Business Model Architecture

### Core Model: Real-Time Fashion Marketplace
- Sources from ~6,000 factories mainly in Guangzhou/China
- More marketplace than brand: pushes production/inventory costs to factory network
- Resembles platform economics more than traditional retail

### Small Batch, On-Demand Production
- New styles produced in quantities as low as 100-200 units
- Scaled up only if real-time sales data justify it
- ~2,000 new SKUs daily, many in "testing mode" (100 items per SKU initial order)
- Design-to-distribution cycle: under 7 days

### AI-Driven Analytics
- Leverages AI to predict trends from social media, search data
- Real-time demand signals drive production decisions
- Ships to 150+ countries
