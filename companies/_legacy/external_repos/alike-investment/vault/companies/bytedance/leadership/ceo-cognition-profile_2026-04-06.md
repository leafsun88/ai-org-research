# CEO认知档案：张一鸣 + 梁汝波

## 1. 张一鸣 — 创始人/战略大脑

### 1.1 基本画像

| 维度 | 内容 |
|------|------|
| 出生 | 1983年，福建龙岩 |
| 教育 | 南开大学软件工程 |
| 创立ByteDance | 2012年 |
| CEO任期 | 2012-2021 |
| 现状 | 退出日常管理，聚焦长期战略/文化/社会责任 |

### 1.2 认知特征

**原生性评估: 极高（5/5级别）**

张一鸣的组织思考是中国科技创始人中最具原创性的：

- **信息效率作为组织设计第一性原理**: 扁平化不是因为"开明"，而是因为"信息流通效率最优"
- **延迟满足(Delay Gratification)**: 不是口号而是贯穿产品/组织/个人的核心操作系统
- **"平庸有引力，你需要逃逸速度"**: 对组织惰性的深刻理解
- **Context not Control**: 与Netflix的Informed Captain高度相似，但独立推导得出
- **AI-first公司定义**: 2015年就将ByteDance定位为AI公司而非内容公司

**关键认知框架**:
1. 公司最强大的敌人是缺乏坦诚
2. 组织扁平是数学推导结果，不是道德选择
3. "务实浪漫主义"——同时持有理想主义和务实态度
4. "Bring Outside In, Avoid Managing Up"——防止大公司病
5. 全面拥抱AGI（2023年公开信）

### 1.3 自我认知

**极高**: 承认自己"缺乏管理技能"并主动退出CEO——这在中国科技创始人中几乎没有先例。选择大学室友梁汝波接任不是"找傀儡"，而是对互补性的深度理解。

### 1.4 与Reed Hastings的对比

| 维度 | 张一鸣 | Reed Hastings |
|------|--------|---------------|
| 组织思考原生性 | 极高（信息论推导）| 极高（文化即战略）|
| 核心机制 | Context not Control | Informed Captain |
| 考核哲学 | OKR+扁平化 | Keeper Test+无奖金 |
| CEO传承 | 主动退出→室友 | 主动退出→双CEO |
| 自我认知 | 极高 | 极高 |
| 全球化视野 | 极强 | 极强 |

**结论**: 张一鸣是与Reed Hastings最接近的CEO认知级别。

## 2. 梁汝波 — 现任CEO/Operator

### 2.1 基本画像

| 维度 | 内容 |
|------|------|
| 关系 | 张一鸣大学室友+联合创始人 |
| 背景 | 研发、飞书、效率工程、HR |
| 类型 | 运营型CEO（Operator） |
| 强项 | 管理、组织建设、法务、内部沟通、后勤 |

### 2.2 认知评估

**原生性评估: 中（3/5级别）**

梁汝波是优秀的operator但不是visionary：
- 能维持张一鸣建立的组织系统
- 缺少做出"TikTok级别"战略判断的能力
- "Corporate housekeeper"的定位准确反映了他的核心能力

**关键风险**: ByteDance的下一个十年需要visionary级别的战略判断（AI时代的platform shift），梁汝波是否具备这种能力？

## 3. CEO传承评估

### 张一鸣 → 梁汝波
- **传承类型**: 主动退出+系统性交接（最佳传承模式）
- **质量**: 高（3年+过渡期，权责清晰）
- **核心风险**: Visionary→Operator的必然能力衰减
- **与Netflix对比**: 极度相似——两者都是创始人主动退出+系统性选择继任者

## 4. 信息缺口

- ⛔ 梁汝波在AI战略上的直接公开表态较少
- ⛔ 张一鸣退出后的实际影响力范围不明
- ⚠️ 内部决策链条（梁汝波vs张一鸣vs Shou Chew）的权力动态不透明
