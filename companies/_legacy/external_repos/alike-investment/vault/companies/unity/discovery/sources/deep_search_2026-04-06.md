# Unity Technologies (U) Deep Search

## 1. 公司概况+商业模式+竞争格局

**核心业务**: Unity是全球领先的实时3D内容创作引擎，同时运营AI驱动广告平台（Vector）。
- Unity Engine: 游戏/非游戏（汽车、建筑、影视）实时3D内容创建平台
- Grow Solutions: AI驱动广告变现平台，核心是Vector AI引擎
- 2024-2026战略重心: 从"大而全"向"高margin AI广告+核心引擎"转型

**商业模式转型**:
- 退出ironSource Ads Network（2026.4.30关闭）
- 出售Supersonic游戏发行业务
- 出售中国区业务（>$1B）
- 聚焦Vector AI广告平台 + Unity Engine
- 目标: 高margin、轻资产、AI-first

**竞争格局**:
- vs Unreal Engine (Epic): 游戏引擎直接竞争，Unity在移动端占优
- vs AppLovin: 移动广告变现直接竞争，Vector vs AXON
- Unity独特优势: 引擎内嵌数据采集，70%头部移动游戏使用Unity
- Developer Data Framework: 90%+ Unity 6.2新项目采用

## 2. CEO/创始人背景+认知+访谈

**CEO Matt Bromberg** (2024.5上任):
- 职业经历: AOL → EA → Major League Gaming → Zynga COO → Unity CEO
- 核心能力: 公司turnaround专家（AOL/Zynga/Unity三次turnaround经历）
- 管理哲学: "执行+问责文化"，重视"how things are done, not just what"
- 上任时诊断: Unity"at war with its customers"（指2023年runtime fee风波）
- 策略: 先修复客户关系 → 精简业务 → 聚焦AI → 提升利润率

**关键认知**:
- 理解平台价值: 引擎数据是广告优化的独特壁垒
- turnaround经验丰富但非创始人CEO
- 职业经理人思维，非技术创始人驱动

## 4. 技术壁垒+AI策略

**Vector AI平台**:
- Unity自研AI广告优化引擎
- 2025 Q1部署后，iOS/Android安装和应用内购提升15-20%
- 2026.1月收入同比增长72%，创历史新高
- 目标: 2026年底达到$1B+年化收入

**数据壁垒**:
- Unity Engine运行在70%头部移动游戏中
- Developer Data Framework提供隐私合规的行为数据
- 引擎-广告闭环: 开发者使用Unity → 产生用户行为数据 → 喂养Vector AI → 提升广告效果
- 这是AppLovin没有的独特优势（无引擎层数据）

**AI应用**:
- 引擎内AI辅助工具
- Vector AI广告优化
- Unity IAP跨平台内购管理
