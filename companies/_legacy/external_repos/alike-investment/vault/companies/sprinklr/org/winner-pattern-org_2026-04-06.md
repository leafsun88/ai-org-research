# Sprinklr — D1-D7 组织穿透

## Step 0: 业务本质画像

**业务本质**: 企业级统一CXM平台（社交+客服+营销+分析）
**理想组织形态**: 产品驱动 + 大企业销售 + AI赋能 + 全球化运营
**关键能力要求**: 产品整合 + 企业销售 + 实施服务 + AI研发

---

## D1: CEO认知质量 — 2.5/5

**依据**:
- 创始人Ragy Thomas有大愿景（Unified-CXM）
- 但Co-CEO模式失败，最终被Board替换
- 新CEO Rory Read是turnaround specialist，非增长型CEO
- Ragy Thomas大量减持 = 创始人信心不足

**核心问题**: 创始人被请走 = D1层面的最大负面信号

## D2: Key Leader Pipeline — 2.0/5

**依据**:
- 创始人+联合CEO同时退出 = 领导层真空
- Rory Read上任仅5个月，团队稳定性未知
- 大公司background的CEO可能带来大公司文化

**不足**:
- Co-CEO失败 = 领导层此前存在分歧
- 核心高管团队信息不足
- 创始人退出后的知识传承风险

## D3: 组织架构适配 — 2.5/5

**依据**:
- 多产品线（社交/客服/营销/分析）需要矩阵型组织
- Gartner/Forrester认可 = 产品能力基础好

**不足**:
- CAC payback 107个月 = 组织效率极低
- 人均营收~$214K = 行业底部
- 产品可能over-engineered → 销售实施周期过长

**不足**:

## D6: Talent Density — 2.5/5

**依据**:

**不足**:

- CEO更换可能引发人才流失
- 创始人大量减持 → 股权激励吸引力下降

## D7: Key Bet质量 — 2.0/5

**依据**:
- AI-native CXM升级是唯一可能的突破点
- Gartner/Forrester评级为产品能力背书

**不足**:
- 没有一个清晰的"big bet"定义公司未来
- turnaround CEO通常专注成本削减而非创新
- $200M回购 = 资本配置偏向回报而非增长
- AI在CXM中不是差异化因素（竞争对手都在做）
