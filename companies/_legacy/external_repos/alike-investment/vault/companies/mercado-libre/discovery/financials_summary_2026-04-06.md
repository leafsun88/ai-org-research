# Mercado Libre (MELI) 财务基本面汇总

## 收入趋势

| 年度 | 收入 | YoY增速 |
|------|------|---------|
| FY2022 | ~$10.5B | — |
| FY2023 | ~$14.5B | +38% |
| FY2024 | ~$20.8B | +43% |
| FY2025 | $28.9B | +39% |

**趋势观察:** 持续高增长（~40% CAGR），FY2025 Q4加速至+45% YoY。增长动力来自电商渗透率提升+Fintech扩张+广告业务爆发。

**Q4 2025详情:**
- Q4营收$8.76B（+45% YoY）
- 运营利润$889M
- 净利润$559M
- TPV $83.7B（+42% YoY）
- GMV $19.9B（+37% YoY）

## 利润率

| 指标 | FY2025 |
|------|--------|
| 营收 | $28.9B |
| 净利润 | $2.0B |
| 净利润率 | 6.9% |
| EBITDA | $4.0B |
| EBITDA Margin | ~14% |
| EBITDA增速 | +24% YoY |
| EPS | $39.40 |

**利润率评估:** 净利润率6.9%偏低，主要因为$13.2B/年的持续投资（物流、人员、技术）。EBITDA margin ~14%更能反映经营效率。但与PDD的25%+净利润率差距显著。关键问题是：利润率是"暂时被投资压制"还是"结构性偏低"？

**评估:** MELI营收规模已达PDD的~70%，但净利润率差距约3.6x。人效差距更为惊人——MELI每员工收入仅$234K（大量物流员工），PDD人效$3M+。两者商业模式差异（MELI重资产物流 vs PDD轻资产平台）是利润率和人效差距的主因。
