# Criteo (CRTO) — Deep Search

## 公司概况
- **总部**: 法国巴黎，Nasdaq上市 (CRTO)
- **业务**: Commerce Media平台，连接营销者和媒体主
- **员工**: ~3,000人（全球29个办公室）
- **市值**: ~$1B（股价大幅下跌后）

## 商业模式
- **Performance Media**: 传统重定向广告业务，贡献大部分收入
- **Retail Media**: 零售商自有流量变现（第一方数据），是战略转型方向
- **Commerce Go**: 自助式广告投放工具（计划2026年Q1上线）
- 收入模型：按广告效果付费（CPC/CPA），TAC（流量获取成本）占比高

## CEO变更（重大事件）
- **Megan Clarken** (2019-2025.2): 前Nielsen CCO，推动从retargeting到commerce media转型
- **Michael Komasinski** (2025.2至今): 前dentsu/Merkle CEO，AdTech老兵，20年+经验
  - 战略重点：Agentic Commerce + AI + Commerce Go自助平台
  - 背景：Vanderbilt工程+哲学本科，Indiana MBA

## 财务（FY2025）
- 营收: $1.9B
- 毛利: $1.0B
- Contribution ex-TAC: $1.2B
- Adjusted EBITDA: $407M (margin ~35%)
- 净利润: $149M
- 自由现金流: $211M
- 股票回购: $152M + 新授权$200M

## FY2026展望
- Contribution ex-TAC: 持平至+2%（constant currency）
- Retail Media: 预计下降mid-to-high teens（重大客户流失）
- Performance Media: 预计mid-single digits增长
- Adjusted EBITDA margin: 32%-34%（较2025年35%下降）
- CapEx: ~$190M（数据中心续约+AI基础设施）

## 关键风险
1. **Retail Media客户流失**: Uber Eats和Target/Roundel退出，$25M收入消失
2. **Cookie deprecation**: 虽然Chrome延迟，但长期趋势不利传统retargeting
3. **CEO更换过渡期**: 新CEO刚上任，战略执行存在不确定性
4. **股价暴跌**: 1年total shareholder return -58.6%
5. **竞争激烈**: 与Meta/Google的广告业务直接竞争

## AI/技术
- **Agentic Commerce**: 嵌入AI agents + MCP协议到平台
- 推荐引擎在离线测试中较baseline LLM提升60%
- 2026 guidance不包含Agentic Commerce收入贡献（太早期）

## 竞争格局
- vs Meta/Google: 开放互联网的独立广告技术
- vs The Trade Desk: 同为DSP但Criteo更偏commerce/retail media
- vs Amazon Ads: Amazon在retail media领域压倒性优势
