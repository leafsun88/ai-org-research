# 11:11 Media Creator Academy / V8 Alpha Live — Source Note

## 面向作者和创作者的频道已经把 Midjourney 从默认推荐降级为“需要重新证明”

这场直播的开场给了一个清晰的外部创作者视角：Midjourney 曾经是作者和写作者最爱用的创意图像工具，因为它很早就提供了独特、想象力强的视觉结果；但主持人也直接说，AI 速度太快，Midjourney 已经落后到他此前“不太能推荐”。V8 Alpha 的意义就在这里：不是在存量粉丝里多一个版本，而是在培训/课程/创作者服务渠道里重新争取推荐位。

“很多作者非常喜欢 Midjourney，甚至有点不健康地喜欢它，因为它是最早真正有创造力、独特的 AI 图像和视频生成工具之一；你把 prompt 放进去，它会做出很惊人的东西。但老实说，因为 AI 进展太快，Midjourney 真的落后了，已经到了我没法推荐它的地步，所以 V8 Alpha 对我来说是个很大的消息。” —— Ben Richardson, 11:11 Media Creator Academy, 2026-04-21

## 主持人把 V8 放在 Microsoft、Sora、Nano Banana 的同一竞争叙事里

这条 source 有一些未经证实的行业猜测，不能直接当事实写入报告，但它很有用地反映了创作者社区如何理解竞争格局：OpenAI 视频、Microsoft image、Nano Banana、Midjourney V8 被放在一个不断替代的工具篮子里。对 Midjourney 来说，外部用户不一定准确区分模型来源或资本关系，他们只关心哪个工具更快、更便宜、更可用，这会把品牌叙事从“艺术模型”拖进“综合创意生产堆栈”的比较。

“如果 Microsoft 已经和 OpenAI 有关系，又和 Midjourney 有关系，而 Midjourney 正在推出 V8 Alpha，那他们为什么还需要另一个新的 image model？它到底是完全新模型，还是在 playground 里测试什么东西？我们现在没有答案，但这个问题本身很有意思，因为这些工具都在争同一批创作者的注意力。” —— Ben Richardson / Bree Bolton, 11:11 Media Creator Academy, 2026-04-21

## V8 Alpha 的 onboarding 把 personalization 变成新用户进入模型前的强制门槛

直播中主持人在第一次生成时被要求先做 personalization ranking，这个细节比看起来重要。Midjourney 不是让新用户直接 prompt，而是让用户先选择一批喜欢的图来解锁 profile；这能提高个人化质量，但也增加首次使用摩擦。对于 creator academy 这种教学场景，主持人现场吐槽“这成了 tutorial”，说明 Midjourney 的 personalization 策略需要在质量和上手速度之间做更好的产品解释。

“它让我先 unlock personalization，也就是必须选一堆我喜欢的照片；所以如果你是 Midjourney 新用户，这就是你要先经历的流程，否则它会让你用 V7。它会给你很多图让你选，我只是在看质量，不一定看内容，选到最后才 profile unlocked。” —— Ben Richardson, 11:11 Media Creator Academy, 2026-04-21

## V8 的图像速度很有冲击力，能重新给课堂和 live demo 带来“现场生成”的可信度

直播里一个正面信号是速度。主持人在 V8 Alpha 里提交 16:9 post-apocalyptic highway 等 prompt，生成四张图时明显惊讶“extremely fast”。对 Midjourney 产品策略而言，速度不只是 compute 指标，它决定了 live 教学、prompt 试错、客户共创和 creator streaming 是否顺滑。V8 Alpha 即使不完整，默认速度已经让用户愿意现场演示，这对社群传播有价值。

“好吧，这太快了，真的非常快；如果你熟悉 Midjourney，你会看到它一次给四张图，然后你可以直接做 subtle variation、strong variation、animate、heart，或者点进去继续 edit。单从这次 post-apocalyptic highway 的生成看，速度是很扎实的。” —— Ben Richardson, 11:11 Media Creator Academy, 2026-04-21

## V8 的写实人物有局部强项，但焦点和 uncanny valley 仍会影响商业可用性

在 close-up 人脸测试里，主持人同时指出了好坏两面：皮肤毛孔、汗珠、细小毛发、虹膜细节很自然，但部分图的眼睛焦点不准，另一些图有 uncanny valley 或塑料感。这是一条很适合进入产品质量判断的证据：V8 已经能给出比旧模型更真实的局部纹理，但商业图片需要稳定命中焦点、解剖和人物真实感，不能只靠四张里挑一张。

“这张脸有些地方做得很好：毛孔、皮肤上的汗、甚至嘴唇上方自然的小绒毛都很真实；但问题是焦点落在鼻子和脸颊上，眼睛反而有点软。另一张眼睛是清楚的，汗珠和雀斑也很好，可你仍然会看到有些图进入 uncanny valley，这就是它现在强和弱同时存在的地方。” —— Ben Richardson / Bree Bolton, 11:11 Media Creator Academy, 2026-04-21

## Midjourney 的 weirdness 参数仍然是差异化资产，但也暴露了用户需要理解参数语义

主持人现场把 weirdness 拉高，结果从可用图像迅速跳到噩梦级结果。这个片段说明 Midjourney 和更 LLM 化的图像工具不同：它仍保留很多模型参数，让用户用 stylization、weirdness、variety、raw 等维度操控结果。对 power users 这是创造力入口；对新用户则是学习成本。Midjourney 的用户教育如果跟不上，参数越丰富越可能变成误用和失望。

“我把 weirdness 拉得太高以后，结果就变得很可怕；把 weirdness 降到零之后，马上好多了。你能看到同一个 close-up prompt，参数一变，出来的东西就从 nightmare 变成可用的人像，所以这些 slider 不是装饰，用户必须知道它们会怎样改变结果。” —— Ben Richardson / Bree Bolton, 11:11 Media Creator Academy, 2026-04-21

## 图像生成已经明显强于视频，V8 Alpha 还没有把 Midjourney 带进完整 production flow

直播里的视频测试是负面证据。主持人把一张 rocking chair 图片做 animate，并用 prompt 要求暗色人物进入画面又消失，但结果没有可靠执行，动作不自然。这个结论和其他 Fast Hours source 互相印证：Midjourney 在图像上仍能打，但视频尚不可交付。它如果要从 image-first 工具扩展到视觉生产平台，视频和 edit/reference 必须从演示功能变成可控工作流。

“图像方面我能看到提升，甚至能拿到一些很好的写实结果；但视频这边，这次测试完全不可用。我们让一个黑影走进画面坐到摇椅上再消失，结果没有按 prompt 来，动作也很不自然。以现在这个阶段看，Midjourney 的 video 对我来说还不能用于交付。” —— Ben Richardson, 11:11 Media Creator Academy, 2026-04-21

## 低价入口有利于试用，但 credit 和 720p/1080p 分层会被拿来和替代工具比较

主持人在结尾提醒用户 V8 Alpha 不是免费，10 美元档给 200 images 和 720p video，30 美元档给更多图片和 1080p video。这个价格信息不应孤立看，而要放在创作者工具替代中：他们会把 Midjourney 和 Kling、Nano Banana、Flux、ChatGPT image、Runway 等放在一起算。Midjourney 的优势如果只是“漂亮”，价格/credit/分辨率会更敏感；如果能成为完整 workflow，订阅阻力会低很多。

“这个不是免费的，你要注册 10 美元一个月，它给你 200 张图片和 720p 视频；30 美元一个月的版本会给更多图片和 1080p 视频。你可以进去玩，但你也有 Kling、Nano Banana、Flux、GPT 这些别的选择，所以要不要为这 200 张图付费，就看你觉得它值不值。” —— Ben Richardson, 11:11 Media Creator Academy, 2026-04-21
