# Midjourney 官方源补充笔记

## 核心来源

- Midjourney careers: https://www.midjourney.com/careers
- Midjourney docs home: https://docs.midjourney.com/hc/en-us
- Midjourney plans: https://docs.midjourney.com/docs/plans
- Midjourney terms: https://docs.midjourney.com/hc/en-us/articles/32083055291277-Terms-of-Service
- Midjourney updates: https://updates.midjourney.com/

## 已核验事实

- Midjourney 在 careers 页把自己定义为一个 community-backed research group，而不是传统 AI lab；核心表述是做能让人 dream bigger、become more capable 的工具。
- 官方 careers 页披露组织工作方式：features 由 community suggested and prioritized，projects 由 engineers and founder lead，团队相比社区和野心“strikingly small”，并且有意维持这种状态以保持快速迭代。
- 官方 careers 页披露公司有 San Francisco、London、New York 办公室，但起源是 remote；大量沟通仍在 Discord；员工分布全球。
- 官方 careers 页披露三类核心工作域：research、infrastructure、frontend；同时存在 image generation 之外的 secret projects。
- 官方 plans 页披露四档订阅：Basic / Standard / Pro / Mega，月价分别为 $10 / $30 / $60 / $120，年付折扣约 20%。
- 官方 plans 页披露：Standard/Pro/Mega 有无限图片 Relax；Pro/Mega 有无限图片与 SD Video Relax；Stealth Mode 只在 Pro/Mega。
- 官方 plans 页披露：年收入超过 $1M 的公司若要拥有生成资产，需要订阅 Pro 或 Mega。
- 官方 terms 有效日期为 2026-02-12；服务通过 midjourney.com、Discord servers 或其他方式提供。
- 官方 terms 把服务定义为持续变化的系统，不保证 quality / stability / uptime / reliability；用户不能依赖服务或生成资产的固定属性。
- 官方 terms 披露默认开放社区与 remix 逻辑：公开场景中的内容默认可见、可 remix；Stealth 是 Pro/Mega 权益但仍存在共享空间限制。
- 官方 community guidelines 禁止成人内容、gore、政治竞选生成、欺诈误导、非法活动等。

## 研究含义

- Midjourney 的组织语言非常清晰：它把 community 当成产品路线图、审美反馈和用户教育的核心基础设施。
- 商业化不是企业销售驱动，而是订阅梯度 + 商业使用门槛 + Stealth 权益；这是“轻 GTM”的模型。
- 官方 terms 对服务质量、IP、社区公开性和用户责任的限制很强，说明它还不是 Adobe Firefly / enterprise indemnification 那种企业合规型创意基础设施。
