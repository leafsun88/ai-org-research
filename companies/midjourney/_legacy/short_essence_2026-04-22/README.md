# Midjourney Legacy Short Essence Archive

This folder archives the old short-form `_essence.md` and `_v2_sample_essence.md` files that were left in the formal podcast / YouTube essence directories before the 2026-04-22 source-note rewrite.

## Why archived

The short essence files were useful during early collection, but they are no longer the formal analysis input because many blocks had overly short quotes and less source-level evidence. The formal Midjourney analysis input is now the full `_source_note.md` set.

## Counts

- `podcasts/`: 13 legacy short essence files
- `youtube/`: 26 legacy short essence files
- Formal source notes left in `vault/*/essence/`: 47 `_source_note.md` files

## Use policy

- Use `_source_note.md` files for analysis, synthesis, and reporting.
- Use this archive only for audit/history, or to compare how the previous short essence format differed from the repaired source-note format.
